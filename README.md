# ProAuction

ProAuction is a modern, performance-focused Auction House plugin designed for
Paper 1.21+ servers. It provides a clean GUI experience, safe purchase handling,
and optional MySQL support for large or networked servers.

## Features
- Auction House browse GUI
- My Listings management
- Expired Items claim system
- Safe purchase confirmation flow
- Configurable GUI branding
- Storage backends:
  - YAML (default)
  - MySQL (optional, auto-created tables)
- Built with server performance and stability in mind

## Requirements
- Paper 1.21+
- Vault (recommended)
- Any compatible economy plugin

## Installation
1. Place the plugin jar into `/plugins/`
2. Restart the server
3. Configure files in `plugins/ProAuction/`
